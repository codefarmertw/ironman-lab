# 《Rust 的戰國時代：探索網頁前端工具的前世今生》 iThome 2024 鐵人賽系列文中範例程式集

- [鐵人賽系列文](https://ithelp.ithome.com.tw/users/20169399/ironman/8023)

## Setup

```bash
$ pnpm install
```

## Usage

可嘗試 esbuild 基本相關設定：

```bash
$ pnpm run build
```

可嘗試 esbuild 在 code splitting 的載入順序問題：
```bash
$ pnpm run esbuild:split
```


可嘗試 Chokidar 套件功能：
```bash
$ pnpm run chokidar
```
